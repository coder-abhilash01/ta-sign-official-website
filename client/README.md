# TA SIGN Business Website

A modern and responsive business website built for **TA SIGN**, providing Digital Signature Certificate (DSC), Trademark Registration, ISO Certification, GST, and compliance-related services.

## Overview

This website is designed to help businesses, professionals, advocates, and individuals access certification and registration services through a clean, user-friendly interface.

## Features

* Modern Responsive Design
* Digital Signature Certificate (DSC) Services
* Trademark Registration Services
* ISO Certification Information
* Contact & Inquiry Forms
* Service-Based Landing Pages
* Mobile Friendly UI
* Smooth User Experience
* Professional Business Branding

## Tech Stack

* React.js
* Tailwind CSS
* React Router
* GSAP
* JavaScript

## Pages

* Home
* About Us
* Trademark Registration
* DSC Services
* Contact Us
* Additional Service Pages

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Purpose

The goal of this project is to provide a professional digital presence for TA SIGN and help users easily explore certification, registration, and compliance services.
